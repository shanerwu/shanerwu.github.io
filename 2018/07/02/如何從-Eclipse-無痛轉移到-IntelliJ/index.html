如何從 Eclipse 無痛轉移到 IntelliJ

建立 Project

Project 定義

  • 相對於 Eclipse 的 Workspace (工作區域)
  • IntelliJ vs Eclipse 名詞定義對照
Eclipse IntelliJ IDEA
Workspace Project
Project Module
Facet Facet
Library Library
JRE SDK
Classpath variable Path variable

建立 Project

  1. 開啟 IntelliJ

  2. 選取要建立的 Project 類型 (以 Empty Project 為例)

  3. 輸入 Project 位置與名稱 -> Finish

    完成後 Project 根目錄會長出 .idea 資料夾 (類似 Eclipse 的 .setting 資料夾)

  4. 設定 Project 內容 :
    選取 SDK (指到本機的 JDK)、設置 Project language level

  5. 搭拉

匯入 Maven 專案

  1. 開啟 Maven 視窗
    View > Tool Windows > Maven Projects
    (或 Ctrl + Shift + A 輸入 Maven Projects)

  2. 點選 + 選擇 Maven 專案 (pom.xml)

  3. 搭拉
    (IntelliJ 在每個專案底下會產生一個 .iml 檔，可設定不顯示在專案視窗上)

設定

Intellij IDEA. Hide .iml files

  • File > Settings > Editor > File Types
    • Ignore files and folders 加入 *.iml;*.idea;

Configure Intellij to use default Eclipse Java import layout

  • File > Settings > Editor > Code Style > Java > Imports

    • Class count to use import with ‘*‘: 改為9999
    • Names count to use static import with ‘*‘: 改為9999

    • Import layout 改為如下 (設定成 Eclipse 預設 Import layout，以免與其他人造成衝突)

取消 Console 行數限制

  • 無法透過內建修改
  • JetBrains\IntelliJ_IDEA\bin\idea.properties
    • 將 idea.cycle.buffer.size 改為 disabled ==> idea.cycle.buffer.size=disabled

Properties

  • No more Properties Editor
  • File > Settings > Editor > File Encodings
    • Properties Files > 勾選 Transparent native-to-ascii conversion

取消 Case Sensitive

  • IntelliSense
  • File > Settings > Editor > Code Completion
    • Case sensitive completion > None

自動加入/移除 Import

  • File > Settings > Editor > Auto Import
    • 勾選 Optimize Imports on the Fly
  • 也可以透過安裝 Plugin Save Actions 管理 Import

資料夾隨版控變更顏色

  • File > Settings > Version Control
    • Check “Show directories with changed descendants”

EditorConfig

  • 內建支援，不需任何外掛

Occurrence 顏色

  • File > Settings > Editor > Color Scheme > General
    • Code > Identifier under caret 更改 Error Stripe mark

Plugins

搜尋 & 安裝 Plugins

  • File > Plugins > Browse repositories > Search Plugins > Download and Install

常用 Plugins

  • Save Actions
    • 可以觸發同 Eclipse save actions 相同效果
  • Lombok Plugin
  • CheckStyle-IDEA
  • Auto filling Java call arguments
    • 透過 intention 自動填入方法參數名稱
  • MoreUnit
  • CodeGlance
    • code minimap

快捷鍵

Super 好用，直接輸入關鍵字來找是否有對應的功能

常用

已預設，但是修改會比較好

可能不太常用

未預設，需自訂

使用 Eclipse 預設快捷鍵

  • 建議還是習慣 IntelliJ 預設快捷鍵，能夠使用的快捷鍵較多
  • File > Settings > Keymap

其他小小技巧

  • Negate an expression
    • 用 ! 選方法，會自動於方法最前面加上 !
  • Postfix Code Completion
    • 使用 . 加上 if 區塊

參考資源

  • Migrating From Eclipse to IntelliJ IDEA

環境

  • JDK 1.8
  • IntelliJ IDEA Community Edition 2017.3.4 x64
    • Community 比 Ultimate 少了非常多功能，尤其是 JavaEE
      例如沒有支援 Tomcat (但可以透過安裝 Maven Plugins 使用)
  • IntelliJ IDEA 2017.3.4 x64 (Educational License)
  • Maven Project
